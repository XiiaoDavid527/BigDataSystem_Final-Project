# Student Part Time job Dashboard

## Overview 

This project is a data driven dashboard that helps university students compare part time job opportunities. The dashboard allows users to analyze jobs based on hourly wage, location, category and language requirements.

## Features
- View all job listings
- Filter jobs by category
- Filter jobs by language requirement 
- Compare wages across job categories
- Identify highest paying jobs
- Interactive dashboard built with Streamlit

## Dataset
The dataset contains:

- Job Title
- Company 
- Hourly Wage
- Location
- Category
- Language Requirement

## Installation
Install dependencies:
```bash
pip install pandas streamlit
```

Run the application:
```bash
python -m streamlit run app.py
```
